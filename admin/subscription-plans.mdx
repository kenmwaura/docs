<Warning>
    - If you intend to use Stripe as a payment method. Please set it up first before adding plans.
    - If you already have plans setup, and later on add Stripe as a payment method, please update all plans (You don't need to change anything)
</Warning>

The Subscription Settings section allows administrators to configure the subscription plans available to users. This includes setting plan prices, durations and features. Proper configuration of subscription settings is essential for managing user access and revenue streams effectively.
