The Users Module in the admin section is designed to manage individual user accounts within the SaaS application. This module provides tools for creating, editing, and organizing user accounts, assigning roles, and connecting to new or pre-existing organizations.

<Warning>
    Deleting a user may result in the deletion of their organization if they are the sole member of that organization.
</Warning>
