### NGINX Server setup
A common issue with Nginx is media files not being sent in chats or campaigns. To resolve this, remove the following line from `.conf` file

```
location ~ .*\.(gif|jpg|jpeg|png|bmp|swf)$
{
    expires      30d;
    error_log /dev/null;
    access_log /dev/null;
}
```
